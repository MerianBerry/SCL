PackIndex & scl::pack::PackIndex::submit ( )

Submit this file index for writing.

Note
If this file index's stream isnt opened by the user, scl will attemt to open it itself, using the given filename, while writing this file index.
It is reccomended to let scl automatically open this index, if this index represents a local file.
